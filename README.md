## Table of contents

## General info
This project is simple app to fecth pictures of kittens from placekitten.com API. You can choose up to 999 pictures to see.

## Technologies
Project is created with:
* Expo-cli
* React-Native
* Redux
* Faker.js


## Setup
To run this project, install it locally using npm:

```
$ cd ../kitten-app
$ npm install
$ npm start
```
